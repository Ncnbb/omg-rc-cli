## 起步 && 安装 
```Shell
npm install -g omg-rc-cli
```

### 支持语法
- [x] ES6 | ES7
- [x] SCSS | CSS
- [x] Svg
- [x] React(JSX | TSX)语法转换
- [x] ESlint
- [x] TypeScript
- [x] handlebars
- [ ] 分包
- [ ] 可配置Typescript语法检测
- [ ] 可视化脚手架配置界面

### 使用环境

node v10.x及以上
